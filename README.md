Welcome to the HHE-Kombat! This repository contains various Hybrid Homomorphic Encryption frameworks.

## Getting Started
To get started with this project, clone the repository and navigate to the framework folders listed above.

```bash
git clone https://github.com/anonymous-for-now/HHE-Kombat.git
cd HHE-Kombat
```
