# AES Frameworks

This directory contains various frameworks and implementations focused on the Advanced Encryption Standard (AES) evaluated within the context of Hybrid Homomorphic Encryption (HHE).

## Included Frameworks

Below is a list of the available AES frameworks:

1. **[BtR_framework](./BtR_framework)**: The BtR framework implementation for AES.
2. **[FbT](./FbT)**: The FbT implementation.
3. **[FFT_CBS](./FFT_CBS)**: AES implementation utilizing FFT-based approaches (CBS).
4. **[Fregata](./Fregata)**: The Fregata AES framework.
5. **[Full-LUT](./Full-LUT)**: AES implementation utilizing a Full Look-Up Table (LUT) approach.
6. **[Hippogryph](./Hippogryph)**: The Hippogryph framework for AES.
7. **[Openssl](./Openssl)**: Reference AES implementation using the OpenSSL library.
8. **[P_encoding](./P_encoding)**: AES implementation focused on P-encoding schemes.
9. **[Tinyaes](./Tinyaes)**: Reference implementation utilizing the lightweight TinyAES library.

## Getting Started

To explore or build a specific AES framework, navigate to its respective directory. For example:

```bash
cd BtR_framework
```

Please refer to any internal documentation or source code within these subdirectories for specific build and execution instructions.
